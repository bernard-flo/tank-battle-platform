# 프로젝트 목표

# 구동 환경
* 지금 너는 docker container 내에서 구동되고 있고, 이 PROMPT 내용으로 반복 실행되고 있음을 참고해.
* 가능한 권한으로 시스템의 모든 기능을 사용하도록 해.
* gpu(cuda)도 사용할수 있어.
* 호스트에 영향을 줄수 있는 심각한 보안 문제는 없어야 해.
